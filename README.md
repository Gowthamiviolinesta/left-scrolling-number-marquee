# left-scrolling-number-marquee
A microcontroller-based project that implements a left-scrolling number marquee on 7-segment displays (SSDs).

Connect the 7-segment displays (SSDs) to the microcontroller as per the ssd_display.h configuration.

Open the project in your IDE/compiler (e.g., MPLAB XC8).

Compile and flash the code to the microcontroller.
